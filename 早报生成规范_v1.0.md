# 📡 科创升学早报生成规范 v1.0

## 一、信息来源优先级

### 🔴 第一优先级（必看）- 官方网站
| 类型 | 具体来源 |
|------|----------|
| 教育部官网 | 教育部阳光高考平台、各省市教育考试院 |
| 高校官网 | 复旦、交大、同济、浙大、南大等华东地区985/211 |
| 上海教委 | 上海市教育委员会官网、各区教育局 |
| 竞赛官方 | 白名单赛事组委会官网 |

### 🟠 第二优先级（重点）- 权威升学公众号
**上海及华东地区核心升学号（每日必刷）：**
- 上海家长、上哪学、上海升学
- 菁教圈、科创升学、自主招生在线
- 强基计划、综合评价、三位一体相关公众号

### 🟡 第三优先级（补充）- 主流媒体
- 澎湃新闻、解放日报、文汇报的教育板块
- 各省市日报的教育新闻

## 二、内容筛选机制

### 2.1 地域优先级（从高到低）
1. **上海本地**：所有上海教委、上海高校、上海竞赛（3-5条，核心板块）
2. **浙江/江苏**：紧邻上海，家长关注度高（1-2条）
3. **广东/北京**：全国性标杆政策（1条，有重大更新才放）
4. **其他省份**：除非是全国性影响的政策，否则不放

### 2.2 内容类型优先级
1. **政策变革**：强基、综评、中考高考等政策变化（最优先）
2. **招生信息**：招生简章、报名时间、名额变化
3. **竞赛动态**：白名单赛事报名、获奖名单
4. **教育改革**：课程改革、评价体系变化

### 2.3 时效性要求
- **只收录48小时内**的最新信息
- 超过2天的除非是重大政策解读，否则不发
- 政策刚发布的24小时内是黄金传播期

### 2.4 去重机制
- 同一政策多家媒体报道的，只选最权威来源整合
- 重复的截止提醒（同一赛事不同阶段）只保留最新一条
- 每个新闻只在一个省市板块出现，不重复

## 三、呈现格式规范

### 3.1 头部
```
📡 科创升学早报 · 月日

> 信息来源：X个权威信源 | 精编X条核心资讯

仅收录月日最新

---
```

### 3.2 正文按省市分组
```
### 📍 上海

🔬 标题（一句话，核心信息前置）

第一句话：核心事实，是什么政策/变化

第二句话：影响解读，对家长/学生意味着什么
```

### 3.3 截止提醒（只放上海及周边）
```
⏰ 上海及周边截止提醒：
• 赛事/报名名称 · 月日截止
• 赛事/报名名称 · 月日截止
```

### 3.4 尾部AI引导
```
---

💡 对以上内容感兴趣的家长，可直接把相关关键词在AI中进行搜索和深入讨论~
```

## 四、执行流程（每日自动执行）

```
08:00 开始执行
│
├─ Step 1：抓取信息源
│   ├─ 访问官方网站（教育部、上海教委、各高校）
│   ├─ 抓取核心升学公众号最新文章
│   └─ 提取24-48小时内的所有相关新闻
│
├─ Step 2：筛选与去重
│   ├─ 按地域优先级排序（上海>浙苏>其他）
│   ├─ 按内容类型优先级排序
│   ├─ 去除重复内容，保留最权威来源
│   └─ 精选7-9条核心资讯（上海3-5条，其他2-4条）
│
├─ Step 3：内容改写与分类
│   ├─ 每条新闻提炼2句话：事实+影响
│   ├─ 按省市分组归类
│   └─ 整理截止提醒（只留上海及周边）
│
└─ Step 4：生成Markdown，保存到 科创升学早报/YYYYMMDD.md
       08:10 发送到群里
```

---

## 五、质量标准

1. **真实性**：每条信息都要有可靠来源，不传播小道消息
2. **时效性**：不发超过48小时的旧闻
3. **实用性**：每条都要说明"对上海家长意味着什么"
4. **简洁性**：每条不超过2句话，手机一屏看完
5. **地域聚焦**：80%内容是上海及长三角，20%是全国性重大政策
