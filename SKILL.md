# 📡 科创升学早报生成技能（合并版 v1.3）

## 背景说明

**这是科创升学群专属的早报 skill。**

本群聚焦以下三大方向：
1. **青少年科创培养**：科创教育、STEM教育、科技创新思维培养
2. **综评升学**：综合评价招生、三位一体、强基计划等升学路径
3. **科创赛事**：白名单赛事、青创赛、宋庆龄发明奖等白名单竞赛

**所有内容以这三个方向为核心，每条资讯都要能回答"对群里的家长和学生有什么价值"。**

## 触发方式

当用户说以下内容时，触发此技能：
- "生成明天的早报"
- "今天的科创升学简报"
- "帮我生成X月X日的早报"
- "科创早报"
- "每日升学简报"

## 二、内容筛选机制

### 地域优先级（从高到低）
1. **上海本地**：所有上海教委、上海高校、上海竞赛（2-3条）
2. **浙江**：紧邻上海，家长关注度高（1-2条）
3. **江苏**：紧邻上海，家长关注度高（1-2条）
4. **全国政策**：仅保留与青少年/学生/升学直接相关的政策（5条简要标题）

### 内容类型优先级
1. **政策变革**：强基、综评、中考高考等政策变化（最优先）
2. **招生信息**：招生简章、报名时间、名额变化
3. **竞赛动态**：白名单赛事报名、获奖名单
4. **教育改革**：课程改革、评价体系变化

### 时效性要求
- **只收录48小时内**的最新信息
- 超过2天的除非是重大政策解读，否则不发

### 价值导向筛选（核心原则）
- ❌ **禁止发布**：
  1. 强调"取消竞赛要求""回归纯高考""弱化科创背景"等不利于科创升学价值导向的信息
  2. 友商广告、商业培训推广、机构招生宣传类内容
  3. **与青少年/学生关联度不高的政策**（如科学基金、国考年龄限制、泛就业政策等）

- ✅ **优先发布**：新增科创专业、大类招生、强基计划、综评中科创背景加分、白名单赛事等

### 去重机制
- 同一政策多家媒体报道的，只选最权威来源整合
- 重复的截止提醒（同一赛事不同阶段）只保留最新一条
- 每个新闻只在一个省市板块出现，不重复

## 五、执行流程

```
1. 信息检索
   - 检索范围：教育部阳光高考平台、上海教委、各高校官网、核心升学公众号
   - 时间窗口：仅收录48小时内最新
   - 关键词：自主招生、科技特长生、综评、三位一体、开放日，白名单赛事

2. 四层筛选
   - 第一层：去重过滤 → 对比历史记录，已发过的直接剔除
   - 第二层：受众匹配 → 仅保留家长/学生关心的
   - 第三层：价值导向 → 剔除弱化科创价值的内容
   - 第四层：地域优先级 → 上海>浙江>江苏>全国

3. 内容撰写
   - 选头条：上海本地、影响最广、时效性最强
   - 标准化格式：每条=核心事实+解读
   - 地域分组：上海（2-3条）/浙江（1-2条）/江苏（1-2条）/全国（5条简报）
   - 截止提醒：两列表格

4. 审稿校验
   - 有没有非家长视角的内容？
   - 有没有弱化科创价值的内容？
   - 有没有和历史记录重复的内容？
   - 头条是不是上海本地、影响最广的？
```

## 六、质量标准

1. **时效性**：只收录48小时内的最新信息
2. **实用性**：每条都要说明"对家长/学生意味着什么"
3. **简洁性**：每条不超过2句话
