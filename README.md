# 📡 科创升学早报

专为鸡血家长社群打造的科创与升学资讯早报自动生成系统。

## 📂 目录结构

```
kechuang-dailynews/
├── SKILL.md                    # 核心技能文件（含完整SOP）
├── docs/
│   ├── 信源与SOP_v2.5.md      # 完整信源清单与采集SOP
│   └── 早报生成规范_v1.2.md   # 生成规范（精简版）
└── README.md                   # 本文件
```

## 🎯 核心文件说明

### SKILL.md
**核心技能文件**，包含完整的 SOP 规范。触发方式：
- "生成明天的早报"
- "今天的科创升学简报"
- "帮我生成X月X日的早报"
- "科创早报"
- "每日升学简报"

### docs/信源与SOP_v2.5.md
**完整信源清单**，包含：
- S/A/B级信源分级体系（含公众号名称、官网URL）
- 每日采集SOP（晨间→午间→晚间）
- 重点推送触发标准
- 内容质量控制标准

### docs/早报生成规范_v1.2.md
**生成规范精简版**，包含格式模板和交付标准。

## 📝 内容标准

**地域分组**：上海（2-3条）/浙江（1-2条）/江苏（1-2条）/全国（5条简报）
**时效性**：只收录48小时内的最新信息
**价值导向**：聚焦青少年科创培养、综评升学、科创赛事三大方向
