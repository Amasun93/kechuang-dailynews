## 一、项目概述

### 项目名称
科创升学早报 · 每日简报自动化

### 核心定位
为鸡血家长社群自动生成科创与升学资讯早报。每天扫描 70+ 权威信源，提炼核心资讯，输出标准化 Markdown + 社群转发文案 + 600px 竖版长图。

### GitHub 仓库
https://github.com/Amasun93/kechuang-dailynews

- `SKILL.md`（v1.3）：执行标准规范，内嵌完整 SOP
- `docs/信源操作指南.md`（v2.5）：信源分级清单与采集 SOP，参考资料
- `README.md`：项目说明

## 二、社群背景与用户画像

### 社群定位
科创升学群，聚焦以下三大方向：
1. **青少年科创培养**：科创教育、STEM教育、科技创新思维培养
2. **综评升学**：综合评价招生、三位一体、强基计划等升学路径
3. **科创赛事**：白名单赛事、青创赛、宋庆龄发明奖等竞赛

### 家长画像
- **地域分布**：以上海为主，辐射浙江（杭州）、江苏、深圳、南昌等地
- **核心焦虑**：升学路径规划、科创加分、该准备什么赛事、综评怎么准备
- **决策需求**：报名时间节点、政策变化、学校动态等 actionable 信息
- **认知水平**：有信息获取能力，愿意为孩子教育投入时间精力，但缺乏系统的一手信息源

### 社群核心价值
- 帮家长每天节省 2-3 小时的信息搜集时间
- 把官方通知翻译成"家长能看懂能行动的人话"
- 让家长不错过任何重要的报名/截止节点

## 三、内容标准

### 地域分组（固定结构）
| 板块 | 条数 | 说明 |
|------|------|------|
| 🔴 头条 | 1条 | 当天最重磅、上海本地、影响最广 |
| 📍 上海 | 2-3条 | 上海教委/高校/竞赛相关 |
| 📍 浙江 | 1-2条 | 紧邻上海，家长关注度高 |
| 📍 江苏 | 1-2条 | 紧邻上海，家长关注度高 |
| 🏆 全国政策 | 5条简报 | 青少年/升学直接相关，标题列表 |

### 长图规格
- 宽度：600px
- 水印：无
- 格式：竖版长图

### 时效性要求
- 只收录 **48小时内** 最新信息
- 超过2天的除非是重大政策解读，否则不发

### 价值导向（核心原则）

**❌ 禁止发布：**
1. 强调"取消竞赛要求""回归纯高考""弱化科创背景"等不利于科创升学价值导向的信息
2. 友商广告、商业培训推广、机构招生宣传类内容
3. 与青少年/学生关联度不高的政策（科学基金、国考年龄限制、泛就业政策等）

**✅ 优先发布：**
- 新增科创专业、大类招生
- 强基计划、综评中科创背景加分
- 白名单赛事报名/获奖公示
- 高校开放日、招生简章

### 内容类型优先级
1. 政策变革（强基/综评/中考高考等）——最优先
2. 招生信息（简章/报名时间/名额变化）
3. 竞赛动态（白名单赛事报名/获奖名单）
4. 教育改革（课程改革/评价体系变化）

## 五、交付物标准

每次生成早报必须同时交付三件套：

### 1. Markdown 文档
- 路径：`科创升学早报/YYYYMMDD.md`
- 包含：头条 + 四地域板块 + 截止提醒 + 尾部引导

### 2. 社群转发语
纯文本格式，直接复制使用，3条核心摘要：
```
🔥【X月X日科创升学早报】
1. xxx
2. xxx
3. xxx
❗️整理自70+权威信源，为您带来一手资讯！
```

### 3. 长图
- 规格：600px 宽竖版，无水印
- 格式：图片文件

## 六、执行流程（四层筛选）

```
1. 信息检索
   - 官方：教育部阳光高考、上海教委、各高校官网
   - 公众号：S级信源全覆盖
   - 关键词：自主招生/科技特长生/综评/三位一体/开放日/白名单赛事
   - 窗口：48小时内

2. 四层筛选
   - 第一层：去重过滤 → 对比历史记录，已发过的剔除
   - 第二层：受众匹配 → 仅保留家长/学生关心的
   - 第三层：价值导向 → 剔除弱化科创价值的内容
   - 第四层：地域优先级 → 上海>浙江>江苏>全国

3. 内容撰写
   - 头条：上海本地、影响最广、时效性最强
   - 标准格式：每条 = 核心事实 + 解读（≤50字）
   - 地域分组固定：头条+上海2-3+浙江1-2+江苏1-2+全国5简报

4. 审稿校验
   - 有没有非家长视角的内容？
   - 有没有弱化科创价值的内容？
   - 有没有和历史记录重复的内容？
   - 头条是不是上海本地、影响最广的？
```

```
> 信息来源：X个权威信源 | 精编X条核心资讯
仅收录X月X日最新

```
