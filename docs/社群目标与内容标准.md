# 社群目标与内容标准

## 社群背景

这个社群来自线下讲座，家长已经和大卫老师发生过真实接触，因此不是普通流量群，而是更接近 VIP 家长经营群。

家长画像：
- 以小学家长为主，初中家长为辅，高中较少。
- 主要分布在上海，也包含浙江、杭州、深圳、南昌等地。
- 对科创赛道有基础兴趣，但多数不是专业人士。
- 既有升学焦虑，也认可长期能力培养，不是纯粹“刷竞赛”人群。

核心焦虑：
- 孩子升学路径怎么走？
- 科创能不能加分，能不能形成优势？
- 现在应该准备什么？
- 哪些学校、政策、赛事值得关注？

## 真实目标

早报是运营工具，也是咨询工具。

它的直接目标不是卖课，而是：
- 用及时资讯保持社群活跃和专业感。
- 用解读持续影响家长认知。
- 让家长看到政策、学校、赛事都在强化科技创新素养。
- 让家长意识到科创规划要早做，尤其小学阶段就要开始建立兴趣、项目和表达能力。
- 为后续咨询、活动、课程转化建立信任基础。

早报必须保持公益感，不直接出现课程广告、招生话术、硬转化。

## 内容判断核心

每条候选信息必须能回答：

1. 对小学/初中家长有什么实际意义？
2. 它说明了什么趋势？
3. 家长现在能做什么？
4. 它是否强化“科创是长期升学规划入口”的认知？

不能回答以上问题的信息，即使新，也不一定发。

## 优先级

### P0 必发

- 强基、综评、三位一体、自招/特色招生中的科创相关变化。
- 上海三公、市重点、科技特色校、科技高中发布的开放日、特色项目、科创成果、招生动向。
- 青创、宋庆龄发明奖、ICC、丘成桐中学科学奖、英才计划等科创发明类白名单赛事的重要节点。
- 报名开始、报名截止、初审结果、获奖名单、公示、政策变化。

### P1 优先发

- AI、机器人、信息学等赛道的青少年活动、赛事和政策。
- 上海及长三角青少年科学教育、STEM、人工智能教育、工程实践项目。
- 对小学阶段长期准备有启发的学校实践或区域活动。

### P2 可作为补充

- 大学实验室开放、科普基地活动、科研院所面向青少年的项目。
- 科创社团成果、学生项目展示、区域科技节活动。

### P3 通常不发

- 普通高考资讯。
- 与科创升学无关的纯政策新闻。
- 泛科技、产业、大学科研进展。
- 商业机构招生、培训推广。
- 技术细节过硬、家长难以行动的竞赛材料。

## 语气

早报应像“情报简报”，不是宣传稿。

写法要求：
- 简洁、判断明确、行动导向。
- 不制造焦虑，但提供紧迫感。
- 不说教，不老师腔。
- 不把“科创”讲成万能升学通道，而是讲成长期能力和升学路径的结合点。

## 解读方向

好解读应该落在这些角度：
- 趋势：政策/学校/赛事为什么值得注意？
- 动作：家长现在该查什么、准备什么、关注什么？
- 路径：这和三公、市重点、综评、强基、白名单赛事有什么关系？
- 年龄：小学家长现在做什么，初中家长现在做什么？
- 风险：如果不准备，可能错过什么节点或积累窗口？
- 素质教育：这件事是否说明真实体验、跨学科探究、社会问题观察、表达展示正在变重要？
- 科创培养：这件事如何帮助孩子从兴趣体验走向问题意识、项目积累和成果表达？

弱解读通常只停在“去参加、去打卡、做记录”。正式早报要再往前推一步：说明为什么这些活动会强化素质教育，为什么有利于科创长期培养。
