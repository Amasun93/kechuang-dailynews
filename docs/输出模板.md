# 输出模板

## 每日早报

正文只给家长看，不出现扫描池解释、命中规则、剔除规则、最终结果等内部执行口径。
正文也不要出现 `摘要：`、`核心事实`、`第一句话`、`第二句话`、`新闻一句话摘要`、`具体新闻内容` 等 agent 写作占位标签。
正文结尾不要自动添加 `今日可引导讨论`、`讨论问题`、`行动建议` 等板块；除非用户明确要求互动引导，早报到截止提醒即可结束。
如果用户明确要求互动问题，只在全文最后添加一句总结式收尾。不要单独起 `### 💬 今日互动` 这类生硬板块标题，也不要用“顺手问一句”这类口语开场；优先用 emoji + 加粗总结开头，从扫描记录的“今日最重要趋势”和“家长最该做的动作”提炼，先给一句判断，再问一个低门槛问题，引出家长分享孩子状态或兴趣方向。

每条资讯的实际显示结构是：

如果多个来源同时提到同一事件、同一政策、同一赛事阶段或同一截止节点，合并成一条资讯，来源并列展示；不要为了增加条数拆成重复资讯。
正文和社群转发语都优先落真实数据：日期节点、人数/名额、场次数、项目数、班型数、方向数等，只写来源已明确公开的数字；没有公开数字就不编、不猜、不补脑。

```md
🔬 **标题（一句话，核心信息前置）**

一段新闻内容，2-3行。把核心事实、时间、影响范围和对家长/学生的价值合并写清楚，不加“摘要：”标签，也不拆成“第一句话/第二句话”。

💡 **解读：** 1-2行；要解释这件事如何强化素质教育、真实体验、项目化学习或科创培养。

来源：[主要来源](URL)、[补充来源](URL)
```

## 🔴 头条：标题

一段新闻内容，2-3行，讲清楚发生了什么、什么时候、影响范围，以及它为什么值得家长关注。不要加“摘要：”标签。

💡 **解读：** 1-2行，说明对家长/学生的行动价值，并尽量落到素质教育、真实体验、项目化学习或科创培养。

来源：[主要来源](URL)、[补充来源](URL)

---

### 📍 上海

🔬 **标题（一句话，核心信息前置）**

一段新闻内容，2-3行，说明事实、时间、影响范围，以及对升学路径、科创准备、学校选择或家长行动的意义。不要加“摘要：”标签。

💡 **解读：** 1-2行，避免只写“打卡/记录/关注”，要点出科创培养价值。

来源：[来源名](URL)

## 全国政策

全国政策只保留与青少年/学生/科创升学直接相关的内容。不要发泛科技、大学科研、成人就业、普通高考类信息。
全国政策固定为 **5条简要标题列表**。不要写成长段正文；不要只写 1-2 条就结束。

## 社群转发语

保持公益感，不写课程广告。
如果用户在对话里要直接复制的版本，回复时把转发语放进代码块。
最后一句默认给出“详情见长图/详情看长图”这类提示。
社群转发语默认写 5 条，每一条都必须是有信息量的真实新闻摘要；不允许用“本轮重点”“今天优先保留”这类空话凑第 5 条。
面向微信复制时，标题、每条摘要、结尾之间主动留空行，避免粘贴后挤成一团。

默认工作流：
1. 先写 `YYYYMMDD-科创升学早报.md`
2. 再写 `YYYYMMDD-社群转发语.txt`
3. 再写 `YYYYMMDD-信源扫描记录.md`
4. 最后生成 `YYYYMMDD-科创升学早报.png`

```txt
🔥X月X日 今日科创升学要点：

1. xxx，明确到 X月X日 / X:XX 截止或开启。

2. xxx，明确到 X个班型 / X个方向 / X个项目 / X所学校 等公开数据。

3. xxx，明确到 X个名额 / X个批次 / X场活动 / X类路径 等公开数据。

4. xxx，明确到真实日期节点或公开量化信息。

5. xxx，必须是第 5 条真实新闻摘要，不能写空泛总结句。

❗整合80+信源，整理自考试院、重点学校官网官微、招生网和赛事官网等权威来源，优先帮家长看清真正影响科创升学判断的学校和节点。

📍详情见长图
```
