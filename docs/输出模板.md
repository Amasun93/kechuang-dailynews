# 输出模板

## 每日早报

正文只给家长看，不出现扫描池解释、命中规则、剔除规则、最终结果等内部执行口径。
正文也不要出现 `摘要：`、`核心事实`、`第一句话`、`第二句话`、`新闻一句话摘要`、`具体新闻内容` 等 agent 写作占位标签。
正文结尾不要自动添加 `今日可引导讨论`、`讨论问题`、`行动建议` 等板块；除非用户明确要求互动引导，早报到截止提醒即可结束。
如果用户明确要求互动问题，只在全文最后添加一个自然问题。不要单独起 `### 💬 今日互动` 这类生硬板块标题；优先从扫描记录的“今日最重要趋势”和“家长最该做的动作”提炼，先给一句判断，再问一个低门槛问题，引出家长分享孩子状态或兴趣方向。

每条资讯的实际显示结构是：

如果多个来源同时提到同一事件、同一政策、同一赛事阶段或同一截止节点，合并成一条资讯，来源并列展示；不要为了增加条数拆成重复资讯。

```md
🔬 **标题（一句话，核心信息前置）**

一段新闻内容，2-3行。把核心事实、时间、影响范围和对家长/学生的价值合并写清楚，不加“摘要：”标签，也不拆成“第一句话/第二句话”。

💡 **解读：** 1-2行；要解释这件事如何强化素质教育、真实体验、项目化学习或科创培养。

来源：[主要来源](URL)、[补充来源](URL)
```

```md
# 📡 科创升学早报 · X月X日

> 信息来源：80+权威信源池 | 精编X条核心资讯
> 时效筛选：近48小时内最有价值信息
> 重点覆盖：上海科创升学、长三角综评/赛事节点、近期关键动作

---

## 🔴 头条：标题

一段新闻内容，2-3行，讲清楚发生了什么、什么时候、影响范围，以及它为什么值得家长关注。不要加“摘要：”标签。

💡 **解读：** 1-2行，说明对家长/学生的行动价值，并尽量落到素质教育、真实体验、项目化学习或科创培养。

来源：[主要来源](URL)、[补充来源](URL)

---

### 📍 上海

🔬 **标题（一句话，核心信息前置）**

一段新闻内容，2-3行，说明事实、时间、影响范围，以及对升学路径、科创准备、学校选择或家长行动的意义。不要加“摘要：”标签。

💡 **解读：** 1-2行，避免只写“打卡/记录/关注”，要点出科创培养价值。

来源：[来源名](URL)
```

浙江、江苏沿用同样格式。

## 全国政策

全国政策只保留与青少年/学生/科创升学直接相关的内容。不要发泛科技、大学科研、成人就业、普通高考类信息。

```md
### 🏆 全国政策

- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
```

## 互动问题（用户要求时才添加）

```md
---

💬 顺手问一句：今天最明显的信号，不是“哪个比赛有用”，而是科创资源正在变成孩子长期体验和积累的入口。你家孩子如果先选一个能坚持3个月的小方向，会更偏AI、机器人、工程实践，还是发明创意？
```

## 信源扫描记录

扫描记录是给运营者看的，不放入早报正文。

```md
# YYYY-MM-DD 信源扫描记录

## 本地历史去重

- 已读取文件：
- 去重剔除：
- 允许再次发布的实质更新：

## 扫描范围

- 官方/考试院：
- 学校：
- 赛事：
- 媒体/公众号：
- 内部线索源：

## 命中来源

1. 来源 + 标题 + 链接 + 入选原因

## 剔除记录

1. 标题/来源：剔除原因

## 今日判断

- 今日最重要趋势：
- 家长最该做的动作：
```

## 周报/公众号方向

周报不要堆日报。建议结构：

```md
# 本周科创升学观察

## 一、本周最重要的3个信号

## 二、对小学/初中家庭意味着什么

## 三、下周需要盯住的报名/截止/开放日

## 四、家长可以和孩子做的3个准备动作

## 五、运营备注（可省略）
```
