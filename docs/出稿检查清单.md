# 出稿检查清单

这份清单是 `kechuang-dailynews` 的硬护栏。每次交付早报前都要逐项检查，不能靠“我大概记得”。

## 一、流程检查

- 已读取当前输出目录的本地历史早报并做去重。
- 已扫描官方/考试院/学校/赛事官网，再补媒体、公众号和内部线索。
- 已统计本轮实际打开核对的网页/PDF正文数。
- 已在扫描记录中写明本轮实际核对的网页/PDF正文数量。
- 已提取可核实的日期/人数/名额/项目数/班型数等真实数据，未公开的数据没有擅自补写。
- 已生成社群转发语，且严格为 5 条有价值的真实新闻摘要。
- 已同步生成手机长图 PNG，并检查其适合手机浏览和群转发。

## 二、分区配额检查

- 上海：`2-3` 条。
- 浙江：`1-2` 条。
- 江苏：`1-2` 条。
- 全国政策：`5` 条简要标题列表。

如果某分区确实不够，不要直接消失：
- 扫描记录里写明不足原因。
- 正文可短写“今日未见值得单列的新信号”。

## 三、全国板块格式检查

全国必须用下面这个格式：

```md
### 🏆 全国政策

- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
- **标题——一句话说明对青少年/升学的意义。** 来源：[来源名](URL)
```

禁止写成长段正文，禁止只写 1 条后就结束。

## 四、截止提醒检查

截止提醒必须拆成三类：

- `赛事`
- `综评和自招系统`
- `其他`

每类都先看有没有临近硬节点；没有硬截止时，只有近30天内有公开新节点，才可以保留 `持续跟踪`。

### 提醒区降密规则

- 周期性赛事如果近30天没有公开新内容，不进正文提醒区。
- `持续跟踪` 不是占位符，必须对应近30天内的新通知、报名、名单、规则或时间节点。
- 没有近端动作、也没有近30天更新的事项，留在内部 watchlist 即可，不要堆给家长看。

### 常驻 watchlist

#### 赛事

- 青创 / 青少年科技创新大赛
- 宋庆龄少年儿童发明奖
- ICC 全球发明大会
- 未来工程师
- 明日科技之星

上面这些都只是内部 watchlist，不代表每天都要显示在正文提醒区。

#### 综评和自招系统

- 上海高校综评报名 / 初审 / 校测 / 志愿确认
- 上海初中综合素质评价结果 / 复核 / 名额分配综合评价与统一招生志愿填报
- 浙江三位一体 / 综合评价
- 江苏综合评价 A 类 / B 类

#### 其他

- 高中校园开放日
- 政策性照顾加分
- 上海科技节 / 职业体验 / 区级活动中心关键窗口

## 五、扫描记录检查

扫描记录里必须有：

- 本地历史去重
- 扫描范围
- 命中来源
- 剔除记录
- 今日判断
- 生成结果
- `本轮人工核对网页/PDF正文：X个`
- `扫描记录中写明：本轮人工核对网页/PDF正文：X个`

## 六、最后一眼检查

- 是否把正文写成给家长看的，而不是给运营者看的？
- 是否每条都回答了“对家长/学生有什么价值”？
- 正文和社群转发语是否至少带了 2-3 个可回溯来源的硬数据点？没有公开数字时是否宁缺毋滥？
- 社群转发语是否严格为 5 条真实摘要，且标题、各条摘要、结尾之间已按微信复制场景留空行？
- 是否把全国、浙江、江苏静默压没了？
- 是否把去重当成了“可以删分区”的理由？
- 长图是否已生成，且版式、字号、表格在手机上可读？
